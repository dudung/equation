# matrix unit
It is a matrix with only one nonzero entry with value $1$, e.g.

$$
\mathbf{E}_{43} = \left[
\begin{array}{ccccc}
0 & 0 & 0 & 0 & 0 \newline
0 & 0 & 0 & 0 & 0 \newline
0 & 0 & 0 & 0 & 0 \newline
0 & 0 & 1 & 0 & 0 \newline
0 & 0 & 0 & 0 & 0
\end{array}
\right]
$$

followed by subscripts corresponding to row and column for the position of value $1$.
